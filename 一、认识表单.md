### 一、认识表单

表单的概念：表单是网页中常见的形式，如登录页面，注册页面的信息填写框。

**组成部分**：

​		一般由提示信息、表单控件、表单域组成。

​		表单域是表单的基本体现用于**容纳**表单控件和提示信息。

​		表单控件一般由按钮、输入框、密码框等组成。

​		提示信息用于说明表单控件的填写提示。

### 二、创建表单

表单的创建使用`<form></form>`双标签创建表单域。

**具体用法**：

```html
<form action="url地址" method="提交方式" name="表单名称">
  	<表单控件></表单控件>
</form>
```



#### 1.表单控件极其常见属性：

表单的核心部分：常见控件有`<input>、<textarea>、<select>`

| `<input>`                        | `<textarea></textarea>` | `<select></select>`                |
| -------------------------------- | ----------------------- | ---------------------------------- |
| 表单输入控件（可定义多种表单项） | 定义多行文本框          | 定义一个下拉列表（必须包含列表项） |



#### 2.action属性：

由于表单输入后的数据需要被后台的程序处理。action属性便用于指定接受并处理表单数据的服务器程序的**url地址**(可以填入邮箱地址)

#### 3. method属性：

method属性用于设置表单数据的提交方式，取值`get(默认取值)`和`post`。

#### 4.name属性：

指定表单的名称，用以区分同一页面的多个表单

**具体用法**：

```html
<body>
	<form action="ts@outlook.com" method="post" name="biaodan">
				<input type="text" name="账号表单">
				<input type="password" name="密码表单">
				<input type="submit" name="提交按钮">
	</form>  
	
 </body>
```

注意：dorm标签的属性并不会直接影响表单的显示效果，要想让一个表单有意义，就必须在form标签之间添加相对应的表单控件
